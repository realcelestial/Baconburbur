# Baconburbur

A Hamburbur Plugin!

It adds few more mods:
+ Elevator Kick Gun
+ Elevator Kick On Touch
+ Send Elevator to Ghost Reactor
+ Destroy Gun
+ Destroy All

# How to install?

- Ok, first, download Hamburbur and GorillaNotifications in the Hamburbur discord Server [ https://discord.gg/75zydB2F3m ]
- Then, put those 2 dlls into the Gorilla Tag Plugins Folder [ Gorilla Tag => BepInEx => plugins ]
Now the Baconburbur Part! 🥓
- Easy, only download the Baconburbur dll and drag it into the plugins folder (with Hamburbur and GorillaNotifications)

  # You are ready to go! Enjoy! 🥓
